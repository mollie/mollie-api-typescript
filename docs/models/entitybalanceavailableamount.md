# EntityBalanceAvailableAmount

The amount directly available on the balance, e.g. `{"currency":"EUR", "value":"100.00"}`.

## Example Usage

```typescript
import { EntityBalanceAvailableAmount } from "mollie-api-typescript/models";

let value: EntityBalanceAvailableAmount = {
  currency: "EUR",
  value: "10.00",
};
```
