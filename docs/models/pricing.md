# Pricing

## Example Usage

```typescript
import { Pricing } from "mollie-api-typescript/models";

let value: Pricing = {
  description: "The Netherlands",
  fixed: {
    currency: "EUR",
    value: "10.00",
  },
  variable: "0",
  feeRegion: "other",
};
```
