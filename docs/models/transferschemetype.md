# TransferSchemeType

The transfer scheme to be used for the transfer. The transfer scheme determines the processing time and method of the transfer.

## Example Usage

```typescript
import { TransferSchemeType } from "mollie-api-typescript/models";

let value: TransferSchemeType = "sepa-credit-inst";
```
